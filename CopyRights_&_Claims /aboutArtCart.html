ArtCart - Complete E-Commerce Website

ArtCart is a fully functional, responsive e-commerce platform crafted using HTML, CSS, and Vanilla JavaScript. It provides a smooth and interactive shopping experience with dynamic product rendering, cart management, and modular design.

📦 Key Features:

  • Dynamic product loading from category HTML files using loadProducts.js
  • Real-time search bar to filter visible products
  • Client-side Add to Cart and Remove logic using localStorage
  • Dedicated Cart Page with quantity display and remove functionality
  • Mobile responsive design using CSS media queries
  • Tooltip and address management via custom JavaScript

🛠 Technologies Used:

  • HTML5 – Semantic structure and layout
  • CSS3 – Component styling and responsive design
  • JavaScript – Dynamic DOM manipulation and logic
  • Google Sheets Integration (Optional) – Using Apps Script for form data storage

📁 Project Structure:

  • index.html – Homepage with product categories
  • cart.html – Cart display page
  • loadProducts.js – Loads category-wise products dynamically
  • cardScript.js – Handles cart addition logic
  • cartRemove.js – Manages cart item removal
  • searchScript.js – Implements live search
  • Multiple CSS files for header, footer, layout, and responsiveness
